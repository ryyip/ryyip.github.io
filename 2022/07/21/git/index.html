Generate ssh key

该命令将在~/.ssh目录下生成两个文件，私钥id_rsa，公钥id_rsa.pub
ssh-keygen -t rsa -b 4096 -C “leo.ye@gmail.com“
ssh-keygen -t rsa -b 4096
cat ~/.ssh/id_rsa.pub
生成后，将公钥内容复制到 git / bitbucket 代码托管站点
