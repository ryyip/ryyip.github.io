Amazon信息爬虫

技术栈 : Python + Redis + MongoDB + Linux
项目背景：公司业务的发展，需要抓取相应的Amazon，eBay信息进行销量分析和产品选型。
项目开发：系统刚开始由我一人进行负责开发，初步进行技术选型，最终决定选用Python作为主要开发语言，选用Scrapy爬虫框架进行开发，MongoDB作为数据库。项目最初的难点主要在于之前没有接触过Python和MongoDB，先通过初步的学习了解它们各自的特性，然后在实际项目中进行具体的实践，加深理解。其次就是通过这个爬虫项目，理解了爬虫的过程，具体实现的功能如下：1.爬取公司在售产品的用户Review，用户评分，帮助也对客诉原因进行分析，2.给定关键词，爬取Amazon，eBay搜索下的前20的商品信息，帮助业务进行新产品开发的选型。3.爬取销量数据，并数据以报表的形式呈现并导出，节约业务人员整理的时间4~6h。还有其他根据需求分析，进行的页面爬取等等
